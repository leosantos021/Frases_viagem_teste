Como usar este guia

Para usar o áudio: Clique no botão "Ouvir Áudio" de qualquer frase. Em alguns navegadores, você precisará interagir com a página primeiro (como clicar em qualquer lugar) para que o áudio funcione.

Para usar os QR codes: Aponte a câmera do seu celular para o QR code para acessar a tradução online.
